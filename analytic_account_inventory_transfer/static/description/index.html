How to use the module: Analytic account Inventory Operation

Overview

The Odoo default behavior has the limitation on applying the analytic account in the inventory operation which somehow is difficult for accounting perspective to filter it in the accounting report. To make it work for accounting, they need to update the journal entry manually and apply analytic account which is the very tedious task.
This is the reason that we build this module to support this need. Meaning that we can apply analytic account in the inventory operation directly and it will update all the journal entry.

Features
Ability for inventory to apply the analytic account in each transfer.
Ability to apply the analytic account in each transfer anytime you want and update it anytime.
Related journal entry with that transfer will update the analytic account follow the inventory
How it works
It will be availale on youtube soon

Step 1: Go to all the transfer listing, and select the transfer you want to apply analytic account


Step 2: You will see the additional button called Apply Analytic Account. This button is available only if the status of your transfer is on Done state


Step 3: To apply analytic account in that transfer, then click the button Apply Analytic Account. You will see the popup as below


Step 4: Select the analytic account, then click the apply it will save that analytic account to that transfer
