Display Group Security

Overview

In Odoo, you can display all security access for each group but you can't display security access for the inherited groups, This module enable you to display all inherited groups hierarchy,menus,access rights, rules and views.

Features
Ability to display all inherited groups hierarchy for each group.
Ability to display all menus access for each group and it's inherited groups hierarchy .
Ability to display all access rights for each group and it's inherited groups hierarchy .
Ability to display all rules for each group and it's inherited groups hierarchy .
Ability to display all views access for each group and it's inherited groups hierarchy .
Ability to display all inherited groups hierarchy for all main groups.
