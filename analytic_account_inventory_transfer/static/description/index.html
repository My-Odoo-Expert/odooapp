How to use the module: “Analytic account Inventory Operation”

Overview

The Odoo default behavior has the limitation on applying the analytic account in the inventory operation which somehow is difficult for accounting perspective to filter it in the accounting report. To make it work for accounting, they need to update the journal entry manually and apply analytic account which is the very tedious task.
That’s why we build this module to support this need. Meaning that we can apply analytic account in the inventory operation directly and it will update all the journal entry.

Features
Ability for inventory to apply the analytic account in each transfer.
Ability to apply the analytic account in each transfer anytime you want and update it anytime.
Related journal entry with that transfer will update the analytic account follow the inventory
